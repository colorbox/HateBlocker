# Hateblocker

https://hateblocker.herokuapp.com/
はてブのホッテントリから、URLに特定のドメインやタイトルに特定の単語を含んだ記事を除外して表示するアプリです。
禁止ワードの管理にははてブのOAuth認証が必要です。
